# 2026/02/17 - Seed7

## 定義

近年では くじら飛行机さんや Genodian なんかの記事で注目されるようになってきたプログラミング言語 Seed7 ね。

これは、 Jancy と Pharo の翻訳やらメモが一通り揃うまでの繋ぎかな。


## 翻訳開始ってことで。

ハードディスク探せば翻訳途中の資料は出でくるでしょう。

いまはとりあえず Termux から Pandoc をインストールして Seed7のソースコードパッケージを展開後に Pandoc で、

```
pandoc -f html manual.htm -t markdown -o manual.md
```

で、 Markdown へ変換したほうが manual.txt を編集するよりラクです(バージョンアップ時の差分比較にはこちらを使いますが)。
アンカーリンクの変換忘れなどアーティファクトはでますのでしかたないところ。このままだと Markor で作業するのは重すぎるからファイルの分割など手作業で進める必要ありなんだよね。
それが終わったら TexTra に通して機械翻訳後に下訳公開、その後は改訳って流れですかね。あまりスタートアップに時間かけたくないので。

ちなみに Seed7 は Termux でビルドして動かせるそうです(X11など別途必要っぽい気がします)
