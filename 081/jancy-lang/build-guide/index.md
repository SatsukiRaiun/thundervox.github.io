# Jancy ビルドガイド

Jancy はオープンソースプロジェクトとして配布されています。使用するには、ビルドする必要があります。

Jancy には複数の依存関係があるため、ビルドはやや複雑に見えるかもしれません。しかし、恐れることはありません。私たちはこのプロセスを段階的〔  その過程を一歩一歩〕にご案内します。
