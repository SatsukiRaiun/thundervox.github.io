# バンドルフレームワーク

このカテゴリのアイテムは、推奨バンドルパッケージである ` jancy_b` にバンドルされています。したがって、`jancy_b` パッケージから Jancy をビルドする場合は、現在のセクションを読み飛ばしても問題ありません。次のすべての依存関係がすでに含まれており、ビルドプロセス中に自動的にビルドされます。

ただし、スタンドアロンパッケージである `jancy` を使用する場合は、最初にこれらの依存関係をダウンロードしてビルドする必要があります。



* AXL\
   \
   Jancy と Graco は、 AXL を汎用 C++ サポートライブラリとして使用しています。
   \
   AXLは以下を特徴とする軽量 C++ ライブラリである。
   * Java 命名規則
   * デフォルトの文字列渡し規則として非ゼロ終端 UTF-8 文字列によるスライス
   * TLS ベースのエラー処理システム
   * 統合参照カウントモデル
   * 一般に広く普及しているライブラリのラッパー
   * その他……。\
   \
  AXL のソースコードは以下から入手できます。\
   [https://github.com/vovkos/axl](https://github.com/vovkos/axl)

* Graco\
   \
   Jancy はフロントエンドのパーサを生成するために文法コンパイラ Graco を使用しています。\
   \
   Gracoは、以下を特徴とする LL(k) 文法テーブル駆動型のトップダウンパーサによる EBNF 方式の生成器です。
   * 予測可能かつ構成可能な競合解決機構
   * 再ターゲッティング可能なバックエンド (Lua 文字列テンプレート方式)
   * ANYTOKEN のサポート
   * 外部トークン化ループ
   * ルール引数を渡したり返したりするための便利な構文
   * その他……。\
   \
  Graco のソースコードは以下から入手できます。\
   [http://tibbo.com/downloads/archive/graco](http://tibbo.com/downloads/archive/graco)
