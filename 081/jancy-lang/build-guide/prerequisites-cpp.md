# C++ コンパイラ

Jancy は C++ で書かれているので、当然 C++ コンパイラと標準 C/C++ ライブラリが必要になります。 Jancy の最新バージョンは以下でテストされました。

 * Windows
   * Microsoft Visual Studio 2010
   * Microsoft Visual Studio 2013
   * Microsoft Visual Studio 2015

 * Linux
   * GCC++ version 4.8.4
   * Clang++ version 3.5.0

 * Mac OS X
   * Clang++ version 7.3.0

上記のバージョン番号は、参照のためだけに提供〔参考程度〕されています。あなたのマシンの C++ コンパイラのバージョンがこれより高いか、あまり低くない場合でも、心配する必要はありません。ほとんどの場合、Jancy は問題なくコンパイルされます。
